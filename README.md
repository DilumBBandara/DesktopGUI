## INSTALLATION TERMUX :
 
  * apt update && apt upgrade -y
  * apt install git -y
  * apt install python -y
  * apt install ruby -y
  * apt install figlet -y
  * gem install lolcat
  * git clone https://github.com/DilumBBandara/DesktopGUI.git
  * cd DesktopGUI
  * python setup.py
  * python desktop.py
